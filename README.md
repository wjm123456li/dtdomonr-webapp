# dtdomonr-fullstack

A full-stack website project with a React frontend and an Express backend.

## Structure

- `apps/web`: React frontend powered by Vite
- `apps/api`: Express backend API

## Quick Start

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev:web` - start the frontend
- `npm run dev:api` - start the backend
- `npm run dev` - run frontend and backend together
- `npm run build:web` - build the frontend
